Neste curso os alunos deverão se dividir em grupos de 5 a 7 alunos, sugiro buscar membros com perfis diferentes para uma boa discussão e aprendizagem. Cada grupo deverá criar um site para: Descrever os problemas que serão resolvidos numericamente; Explicar os métodos usados; Apresentar gráficos das funções dos problemas (quando possível); Apresentar os gráficos e/ou os resultados das soluções; Discutir os resultados; Deixar o código acessível; Discutir as diferenças entre diferentes métodos; Sempre que possível, usar mais de uma forma de resolver o problema. O site deverá conter o nome de todos do grupo. Fica a critério do grupo a linguagem a ser usada, recomendo usar SciLab, GNU Octave, MatLab e Python. Existem vários exemplos de métodos implementados, PROCUREM e USEM !
